  title: '[Primary Keyword] — [Optional: Free Calculator | Site Name]',
  description: '[120–160 characters. Include primary keyword. State value clearly. Optional CTA.]',
  alternates: {
    canonical: 'https://yourdomain.com/[slug]',
  },
}

## FAQ Formatting Guide

Write FAQ questions in the voice of a user searching on Google:

**Good (matches search intent):**
- "What is a FIRE number?"
- "How much do I need to retire at 40?"
- "What is the 4% rule?"
- "Is a 50% savings rate realistic?"

**Avoid:**
- "Why should I use this calculator?" (promotional)
- "What features does this tool have?" (product-focused)

Each answer should:
- Start with a direct answer to the question (first sentence)
- Add context or nuance (2–3 sentences)
- Link to a related calculator or article where relevant
- Be 50–100 words
